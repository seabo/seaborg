# Seaborg strength-test openings v1

These eight elementary positions were authored for Seaborg from standard
starting-position opening moves. George Seabridge and Seaborg contributors
dedicate this file to the public domain under CC0 1.0 Universal:
https://creativecommons.org/publicdomain/zero/1.0/

The suite is intentionally small so the orchestration is repository-owned and
auditable. It is a versioned test input, not a claim of broad opening coverage.
Changing it requires a new version and updating the checksum embedded in
`strength_test.py`.
